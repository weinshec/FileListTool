FileListTool
============

urwid based console application for easy filelist creation with dq2


## Description

Commanline application for easy creating filelists on the localgroupdisc using dq2 and rucio.

**Dependencies:**
+ python 2.5 or above
+ urwid library
+ dq2-tool ready
